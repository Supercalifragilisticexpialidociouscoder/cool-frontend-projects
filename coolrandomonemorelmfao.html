Examples & copy

Copy these snippets into your project to re-create the left→right dim effect on any card-like element:

/* Add this to the element */
.element{position:relative}
.element::after{
  content:'';position:absolute;inset:0;
  background:linear-gradient(90deg, rgba(0,0,0,0) 0%, rgba(0,0,0,0.2) 40%, rgba(0,0,0,0.6) 100%);
  transform:translateX(-110%);transition:transform .7s cubic-bezier(.18,.9,.3,1),opacity .3s;
}
.element:hover::after{transform:translateX(110%);opacity:1}
